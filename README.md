# lyrics-display
This project is a single page web app that delivers a dynamic and motivational visual experience.

It brings together three main elements:

Fast Moving Starfield: Using the particles.js library, the background becomes a dark, cosmic scene filled with fast moving stars that create a feeling of motion, energy, and endless potential.

Motivational Lyrics: At the center of the screen, inspiring lyrics appear, such as “If I could do it, then why can’t you? Get up and make it work...” to encourage motivation and determination.

Typing Effect: Each line of text is revealed with a typing animation, adding rhythm, anticipation, and focus to the message.

In essence, it’s a full screen, visually captivating display that combines powerful words with a vibrant, limitless backdrop — a digital boost of energy and inspiration.
